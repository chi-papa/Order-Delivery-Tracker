発注・配送メモ
農業資材などの発注・配送状況をブラウザだけで管理できる、シンプルな静的Webアプリです。
サーバー不要・インストール不要。index.html をブラウザで開くだけで動作します。

機能

発注登録 — 発注日・メーカー・商品名・数量・入荷予定日・送り状番号・配送会社・ステータス・備考を一括登録
ステータス管理 — 未発送 / 発送済 / 到着 をワンクリックでサイクル変更
荷物追跡 — trackings.jp 経由で14社以上に対応。送り状番号入力済みの追跡ページを直接開く
備考インライン編集 — テーブル上でクリックしてその場で編集・保存
フィルター＆検索 — ステータスフィルター / 商品名・メーカー・送り状番号のキーワード検索
メーカー管理 — 発注先メーカーの追加・削除（デフォルト9社プリセット済み）
配送会社管理 — 配送会社の追加・削除。trackings.jp IDを設定すると追跡リンクが有効になる
ユーザー名 — ヘッダーに担当者名を表示（複数人での運用を想定）
データ永続化 — すべてのデータはブラウザの localStorage に保存。サーバー通信なし


使い方
index.html をブラウザで開く
以上です。ビルド・サーバー・依存パッケージは一切不要です。
配送会社の追跡IDを追加する
右パネル「配送会社管理」から会社名と trackings.jp ID を登録すると、その会社の「追跡」ボタンが有効になります。IDを空にした場合はGoogle検索にフォールバックします。
